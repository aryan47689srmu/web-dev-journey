# Semantic Tags in HTML

## &#x20;What are Semantic Tags?

Semantic tags in HTML are elements that clearly describe their meaning to both the browser and the developer. They make the structure of a webpage more understandable and meaningful.

Instead of using generic tags like `<div>` and `<span>`, semantic tags tell **what kind of content** they contain.

---

## &#x20;Why Use Semantic Tags?

* Improves **readability** of code
* Enhances **SEO (Search Engine Optimization)**
* Helps **screen readers** and accessibility tools
* Makes website structure **clean and organized**

---

## &#x20;Common Semantic Tags

### 1. `<header>`

Represents the top section of a webpage or a section.

```html
<header>
  <h1>My Website</h1>
  <p>Welcome to my site</p>
</header>
```

---

### 2. `<nav>`

Defines navigation links.

```html
<nav>
  <a href="#">Home</a>
  <a href="#">About</a>
</nav>
```

---

### 3. `<section>`

Represents a section of content.

```html
<section>
  <h2>Services</h2>
  <p>We provide web development services.</p>
</section>
```

---

### 4. `<article>`

Represents independent content (like blog posts).

```html
<article>
  <h2>Blog Title</h2>
  <p>This is a blog post.</p>
</article>
```

---

### 5. `<aside>`

Used for side content (like ads or sidebars).

```html
<aside>
  <p>Related links</p>
</aside>
```

---

### 6. `<footer>`

Represents the bottom section of a page.

```html
<footer>
  <p>© 2026 My Website</p>
</footer>
```

---

### 7. `<main>`

Represents the main content of the document.

```html
<main>
  <h1>Main Content</h1>
</main>
```

---

### 8. `<figure>` and `<figcaption>`

Used for images with captions.

```html
<figure>
  <img src="image.jpg" alt="Example">
  <figcaption>Example Image</figcaption>
</figure>
```

---

## &#x20;Semantic vs Non-Semantic Tags

| Semantic Tags | Non-Semantic Tags |
| ------------- | ----------------- |
| `<header>`    | `<div>`           |
| `<article>`   | `<span>`          |
| `<footer>`    | `<div>`           |

Semantic tags clearly define content, while non-semantic tags do not.

---

## &#x20;Advantages of Semantic HTML

* Better **search engine ranking**
* Improved **accessibility**
* Easier **maintenance**
* Cleaner **code structure**

---

## &#x20;Conclusion

Semantic tags are essential for building modern, accessible, and SEO-friendly websites. They help both developers and browsers understand the structure and meaning of content.

---

## &#x20;Quick Tip

Use semantic tags whenever possible instead of `<div>` to make your HTML more meaningful and professional.
