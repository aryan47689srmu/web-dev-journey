# Images, Lists, and Tables

## Images in HTML

Images are added using the `<img>` tag.

### Syntax:

```html
<img src="image.jpg" alt="Description" width="300" height="200">
```

### Attributes:

* `src` → Image path
* `alt` → Alternative text
* `width` → Width of image
* `height` → Height of image

### Example:

```html
<img src="cat.jpg" alt="Cute Cat" width="200">
```

## Lists in HTML

### 1. Ordered List (Numbered)

```html
<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>
```

### 2. Unordered List (Bulleted)

```html
<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Mango</li>
</ul>
```

### 3. Description List

```html
<dl>
  <dt>HTML</dt>
  <dd>Markup language</dd>
</dl>
```

---

## Tables in HTML

Tables display data in rows and columns.

### Syntax:

```html
<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Aryan</td>
    <td>18</td>
  </tr>
</table>
```

### Tags:

* `<table>` → Table container
* `<tr>` → Table row
* `<th>` → Header cell
* `<td>` → Data cell

---

## Summary

* Use `<img>` to display images
* Use `<ol>`, `<ul>`, `<dl>` for lists
* Use `<table>` for tabular data
