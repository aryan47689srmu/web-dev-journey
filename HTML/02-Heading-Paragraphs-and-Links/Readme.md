# Heading, Paragraph, and Links

## Heading

Headings are used to define titles and subtitles on a webpage.

HTML provides 6 levels of headings:

* `<h1>` (largest and most important)
* `<h2>`
* `<h3>`
* `<h4>`
* `<h5>`
* `<h6>` (smallest)

### Example:

```html
<h1>Main Title</h1>
<h2>Subheading</h2>
<h3>Section Title</h3>
```

---

## Paragraph

Paragraphs are used to display text content.

* Defined using the `<p>` tag
* Automatically adds spacing

### Example:

```html
<p>This is a paragraph in a webpage.</p>
```

---

## Links

Links are used to navigate between web pages.

* Created using `<a>` tag
* `href` attribute defines the destination

### Example:

```html
<a href="https://www.example.com">Click here to visit</a>
```

---

## Summary

* Headings → Structure the content
* Paragraphs → Add readable text
* Links → Connect pages together

These are the foundation of frontend web development.
